# NLP Sentiment Analysis

## Overview
This project analyzes text data and predicts whether sentiment is positive or negative using Natural Language Processing and Machine Learning.

## Business Goal
Sentiment analysis helps businesses understand customer feedback, reviews, and opinions at scale.

## Workflow
- Load labeled sentiment data
- Clean and preprocess text
- Convert text into TF-IDF features
- Train a Logistic Regression classifier
- Evaluate model performance
- Test predictions on new sample sentences

## Tools Used
- Python
- Pandas
- Scikit-learn
- Regular Expressions
- TF-IDF Vectorization
- Logistic Regression

## Key Learning
The model demonstrates an end-to-end NLP workflow. Performance is limited by the small sample dataset and would improve with a larger real-world dataset.

## Output
The project predicts whether new text examples express positive or negative sentiment.
