# Web Scraping and Job Market Analysis

## Overview
This project collects job listing data from a website and analyzes trends in job titles, companies, and locations.

## Business Goal
The goal is to understand hiring trends and identify the most common roles, companies, and locations in the job market.

## Data Collection
- Data scraped from a public job listing site
- Extracted fields:
  - Job title
  - Company
  - Location

## Analysis Performed
- Top job titles
- Top hiring companies
- Top job locations

## Tools Used
- Python
- Requests
- BeautifulSoup
- Pandas
- Matplotlib

## Key Insights
- Certain job titles appear more frequently, indicating higher demand
- Some companies consistently appear, suggesting active hiring
- Job listings are concentrated in specific locations

## Output
The project demonstrates how to collect, clean, and analyze real-world data for business insights.
