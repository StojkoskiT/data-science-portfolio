# E-commerce KPI Dashboard

## Overview
This project analyzes e-commerce sales data to calculate key business KPIs and trends for dashboard reporting.

## Goals
- Measure total revenue
- Measure total orders
- Calculate average order value
- Analyze monthly sales trends
- Compare category and regional performance

## Tools
- Python
- Pandas
- Matplotlib

## Business Insights
- Revenue trends help identify seasonality and changes over time
- Category analysis shows which areas drive most sales
- Regional analysis highlights top-performing markets
- Sub-category analysis reveals more detailed product opportunities

## Output
This project prepares business metrics and visualizations that can be used in a dashboard.
